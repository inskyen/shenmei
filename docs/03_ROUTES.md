# 03_ROUTES：審美者頁面與路由文檔

## 1. 路由設計原則

審美者的頁面應圍繞四個核心對象展開：

```text
video   影片本體
post    採樣動態
module  頻道
profile 使用者
```

首頁和頻道頁展示的主資料應該是 `posts`。
影片主頁 `/v/[id]` 則以 `video` 為中心，沉澱播放、影片資訊、所有推薦與影片評論。

## 3. `/` 大廳

### 定位

首頁大廳是全站入口，預設展示全站最新採樣動態。

它不是某個頻道，也不是影片列表。

### 頂部導航

```text
追蹤 / 最新 / 頻道
```

第一版：

- `最新` 為預設 active。
- `追蹤` 顯示目前使用者追蹤之人的公開採樣動態。
- `頻道` 在首頁內切換至頻道列表，點擊卡片後進入 `/m/[slug]`。

### 底部導航

```text
發現 / 頻道 / 採樣 / 私訊 / 我的
```

當前行為：

- `發現` → `/` 的最新頁籤
- `頻道` → `/` 的頻道頁籤或 `/m`
- `採樣` → `/submit`
- `私訊` → `/messages`
- `我的` → `/u/[current_username]`，未登入則 `/login`

### 主要資料

首頁應查：

```text
posts
profiles
videos
modules
like_count
comment_count
```

### 卡片內容

每張採樣卡片應展示：

- 採樣人暱稱與頭像。
- 發布時間。
- 推薦理由 `post.note`。
- 影片封面。
- 影片標題。
- UP 主 / 作者。
- 所屬頻道。
- 喜歡數。
- 留言數。

### 互動

- 點擊封面：開啟沉浸播放，或後續進入 `/v/[id]`。
- 點擊影片標題：進入 `/v/[id]`。
- 點擊推薦文字或卡片：進入 `/p/[id]`。
- 點擊採樣人：進入 `/u/[username]`。
- 點擊頻道：進入 `/m/[slug]`。

### 空狀態

```text
還沒有採樣動態。成為第一個留下審美痕跡的人。
```

## 4. `/submit` 發佈採樣

### 定位

讓使用者發布一條採樣動態。

發布行為像 Soul 瞬間，但內容是影片採樣。

### 訪問權限

- 未登入：導向 `/login`。
- 已登入：可發布。

### 表單欄位

必填：

- B 站連結或 BVID。
- 推薦理由。

選填：

- 頻道，可多選。
- 未選頻道時，預設發到大廳最新流。

後續可自動解析：

- 影片標題。
- 封面。
- UP 主。

若第一版無法自動解析，可先允許手動填入。

### 校驗

- B 站連結或 BVID 必填。
- 推薦理由必填。
- 推薦理由建議最少 10 個字。

### 寫入流程

```text
檢查登入狀態
→ 解析 BVID
→ 查 videos 是否已存在
→ 不存在則建立 videos
→ 建立 posts
→ 若選擇頻道，建立 post_modules
→ 發布成功後跳轉 `/`
→ 大廳最新流立即展示剛發布的採樣動態
```

### 成功提示

```text
已發布，這支影片有了新的審美註解。
```

### 發布後跳轉

發布成功後回到 `/` 大廳最新流，而不是直接進入 `/p/[id]` 或 `/v/[id]`。

原因：

- 讓使用者立即看到自己剛發布的內容出現在公共大廳。
- 形成「剛發就被收錄」的即時反饋。
- 更接近 Soul 瞬間動態發布後回到信息流的感覺。

## 5. `/v/[id]` 影片主頁

### 定位

以影片為中心的沉澱頁，接近 B 站影片頁的角色。

它展示的不是某一個人的推薦，而是一支影片在審美者裡累積的所有推薦與討論。

### 主要資料

```text
video
posts for this video
profiles for those posts
comments where target_type = video
modules related through posts
```

### 頁面內容

- 影片播放器。
- 影片標題。
- UP 主 / 作者。
- 原始來源連結。
- 推薦過這支影片的人。
- 所有推薦理由列表。
- 影片公共留言區。
- 相關頻道。

### 互動

- 播放影片。
- 對影片留言。
- 點擊某條推薦進入 `/p/[id]`。
- 點擊推薦者進入 `/u/[username]`。
- 點擊頻道進入 `/m/[slug]`。

### 空狀態

如果影片還沒有公共留言：

```text
這支影片還沒有留下公共回聲。
```

## 6. `/p/[id]` 採樣動態詳情

### 定位

展示單篇採樣動態的完整內容。

它關注的是某個人的一次推薦，而不是影片的全部沉澱。

### 主要資料

```text
post
profile
video
modules
comments
reactions
```

### 頁面內容

- 影片播放器。
- 影片標題。
- UP 主 / 作者。
- 採樣人。
- 推薦理由。
- 所屬頻道。
- 喜歡按鈕。
- 留言區。

### 互動

- 喜歡 / 取消喜歡。
- 發表針對這條推薦的留言。
- 點擊影片標題進入 `/v/[id]`。
- 點擊採樣人進入 `/u/[username]`。
- 點擊頻道進入 `/m/[slug]`。

### 留言區空狀態

```text
還沒有人留言。也許你可以寫下第一句回聲。
```

## 7. `/m` 頻道列表

### 定位

展示所有頻道入口。

頻道是審美者的板塊 / 主題空間，不是臨時房間。

### 主要資料

```text
modules
每個 module 的 post_count
每個 module 的最新 posts 預覽，可後續做
```

### 頻道卡片

每張卡片展示：

- 頻道名稱。
- 頻道簡介。
- 封面或主色。
- 採樣動態數。
- 最新更新時間。

### 頻道類型

類型頻道：

- 影像
- 音樂
- 動畫
- 設計

審美氛圍頻道：

- 藍色時刻
- 孤獨夏天
- 深夜放映
- 城市雨聲

產品應更突出審美氛圍頻道。

### 空狀態

```text
頻道正在整理展牆。
```

## 8. `/m/[slug]` 頻道頁

### 定位

展示某個頻道下的採樣動態。

### 主要資料

```text
module
posts
profiles
videos
like_count
comment_count
```

### 頁面內容

- 頻道名稱。
- 頻道簡介。
- 頻道封面 / 主色。
- 頻道內最新採樣動態。
- 發佈到此頻道的入口。

### 子導航

第一版可先只做：

```text
最新
```

後續可加入：

```text
精選 / 熱門
```

### 權限

- 所有人可讀 active 頻道。
- 管理員可建立 / 編輯頻道。
- 使用者可投稿到頻道。

## 9. `/u/[username]` 使用者頁

### 定位

展示某個使用者的審美檔案。

### 當前狀態

已完成使用者資料、審美標籤、統計、追蹤、私訊入口與採樣列表分頁載入。

### 主要資料

```text
profile
posts
follower_count
following_count
```

### 頁面內容

- 頭像。
- 暱稱。
- 簡介。
- 發布過的採樣動態。

- 審美標籤。
- 追蹤與粉絲統計。
- 追蹤按鈕與私訊入口。
- 本人的採樣與編輯資料入口。

### 空狀態

```text
這裡還沒有留下採樣痕跡。
```

## 10. `/login` 登入 / 註冊

### 定位

帳號入口。

### 功能

- Email 登入。
- Email 註冊。
- 六位 Email OTP 驗證。
- 註冊後建立 profile。
- 註冊後自動生成唯一數字型 `username`；目前前端禁止自行修改審美號。

### 文案

- `歡迎回來`
- `申請成為採樣人`
- `信箱`
- `密碼`
- `登入`
- `註冊`

Supabase 設定已統一由環境變數與共用 client 提供。

## 11. `/search` 搜尋

### 定位

首頁右上角搜尋入口對應此頁，用來搜尋 `video` 與使用者。

它不是頻道列表；頻道可由首頁頁籤或底部導航進入。

### 當前功能

- 搜尋站內已收錄影片。
- 搜尋使用者暱稱與數字審美號。
- 點擊影片進入 `/v/[id]`。
- 點擊使用者進入 `/u/[username]`。

## 12. `/messages` 訊息列表

### 定位

私訊入口。

### 當前功能

- 私訊會話列表。
- 最新訊息與相對時間。
- 未讀數。
- 建立新會話。
- 底部導航未讀紅點。

## 13. `/messages/[id]` 私訊詳情

### 當前功能

- 一對一文字私訊。
- Supabase Realtime 即時接收。
- 已讀狀態與未讀數。
- 會話資料預載與返回緩存。

尚未完成：封鎖、檢舉與訊息頻率限制。

## 14. `/notifications` 通知列表

### 定位

通知入口，與私訊分開。

通知包含：

- 有人喜歡我的 post。
- 有人留言或回覆我。
- 有人追蹤我。
- 後續的頻道投稿 / 精選狀態。

### 當前功能

- 喜歡通知。
- 留言與回覆通知。
- 追蹤通知。
- 未讀計數與批量標記已讀。

## 15. `/settings` 設定

### 當前功能

- 暱稱、頭像、簡介與審美標籤。
- 私訊權限。
- 登出。

尚未完成：帳號刪除、登入裝置、安全記錄與通知偏好。
