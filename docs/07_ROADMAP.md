# 07_ROADMAP：審美者路線圖

> 更新時間：2026-07-18

## 1. 當前判斷

審美者已經越過「展示原型」階段，具備可使用的社交產品 MVP。

當前定位：

```text
核心 MVP 完成
→ 種子內測準備完成
→ 公開測試前治理施工中
```

階段評估：

- 核心產品完成度：約 90%。
- 公開運營準備度：約 60%。
- 適合 10～30 位熟人或受邀使用者進行種子內測。
- 暫不適合完全開放註冊並大規模宣傳。

## 2. 已完成階段

### Phase 0：產品與資料模型定義

- 確認 `video + post` 雙核心。
- 完成 PRD、資料模型、路由與 API 初版文檔。
- 確認大廳、頻道、採樣、私訊、我的五個核心入口。

### Phase 1：帳號與資料庫地基

- Supabase Auth、共用 client 與環境變數。
- `profiles / videos / posts / modules / post_modules / comments / reactions`。
- Email / Password 註冊登入與六位 OTP。
- 數字審美號、profile trigger 與 RLS。

### Phase 2：採樣發布閉環

- `/submit` 支援 B站連結與 BVID。
- 影片資料解析、去重與 `videos + posts` 寫入。
- 普通使用者發到大廳；審美者與超管可選頻道。
- 發布完成後返回最新流並立即看到新內容。

### Phase 3：內容雙核心

- `/v/[id]` 影片主頁。
- `/p/[id]` 採樣詳情、播放、喜歡、留言與回覆。
- 採樣軟刪除保留影片本體。
- 已刪除主留言保留其他使用者的回覆脈絡。

### Phase 4：首頁、頻道與搜索

- 首頁「追蹤／推薦／最新／頻道」切換與滑動手勢。
- 最新流分頁載入、預載與返回位置恢復。
- `/m` 頻道列表與 `/m/[slug]` 頻道詳情。
- 頻道建立、關閉、排序與投稿權限。
- `/search` 搜索影片與使用者審美號。

### Phase 5：人與關係

- `/u/[username]` 使用者頁與採樣懶載入。
- 使用者追蹤與追蹤信息流。
- 角色：`member / aesthete / super_admin`。
- 審美者徽章與超管治理權限。

### Phase 6：私訊與通知

- 私訊列表、建立會話與一對一文字訊息。
- Supabase Realtime 即時推送。
- 未讀數與底部導航提示。
- 喜歡、留言、回覆與追蹤通知。

### Phase 7：App 化體驗

- 固定底部導航與頁面預載。
- 沉浸影片播放。
- 詳情頁影片隨捲動縮小。
- 主要页面骨架、緩存與返回狀態保留。
- 品牌蝴蝶圖標、分享圖與行動端主畫面安裝體驗。

## 4. 種子內測門檻

以下條件滿足後，可邀請小範圍使用者：

- 正式環境可正常註冊、登入、採樣、留言、私訊與刪除。
- 最新 migration 已執行並人工驗證 RLS。
- 超管可快速處理惡意內容。
- 已準備至少一個公開聯絡入口。
- 每個主頻道有基本內容，不出現大面積空頁。

種子內測期間主要觀察：

- 註冊到首次採樣的完成率。
- 每日採樣、留言、追蹤與私訊數。
- D1 / D7 回訪。
- 載入失敗、Realtime 斷線與 B站解析失敗。
- 垃圾內容、騷擾與檢舉量。

## 5. 公開測試門檻

公開測試不是「頁面都能點」；至少還要具備：

1. 基本防刷與帳號治理。
2. 使用者可見的檢舉、封鎖與規則。
3. 可追蹤的錯誤與資料恢復能力。
4. 明確的隱私、內容與侵權政策。
5. 一週以上種子內測沒有資料丟失或高頻阻斷錯誤。
