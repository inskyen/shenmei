## 目前產品定案

審美者是一個以影片為入口的審美採樣社交產品。

目前採用 **video + post 雙核心**：

- `video`：影片本體，負責內容沉澱，對應 `/v/[id]`。
- `post`：使用者的一次推薦，負責信息流傳播與審美表達，對應 `/p/[id]`。

第一版核心閉環：

```text
登入
→ 發布 B 站影片
→ 填寫推薦理由
→ 可選頻道，不選則發到大廳
→ 回到大廳最新流看到自己的內容
```

## 已確認決策

- 第一版註冊 / 登入後即可發布，不做邀請制。
- 第一版只支援 B 站影片，但資料結構預留多平台。
- 推薦理由必填。
- 頻道由管理員建立。
- 發布時可選頻道，不選則進入大廳最新流。
- 發布成功後跳回 `/` 大廳最新流，讓使用者看到剛發的內容。
- `reactions` 第一版只做 `like`。
- 首頁右上角搜尋入口進入 `/search`，用來搜尋 `video` 與使用者。
- 首頁頂部與底部 `頻道` 會切換至頻道列表；點擊卡片後進入 `/m/[slug]`。
- 私訊與通知拆分：`/messages` 與 `/notifications`。
- `/u/[username]` 第一版做極簡使用者頁。
- `username` 註冊時自動生成數字，後續可修改。
